CSS Selectors

Types of Selectors

Type Selects Example Code Example
Type Selector Selects the HTML Element p, h1, div, ect p {color: green;}
Universal Selector Selects ALL Elements of any type * * {font-family: Veranda;}
Class Selector Selects the class attribute assigned to it .[attribute name] If a 'p' has class = "brand" -> .brand {font-size= 12;}
ID Selector Selects the attribute name assigned to it #[attribute name] If a 'p' has id = "title" -> #title {font-size=12;}
Attribute Selector Selects HTML Elements tyhat already contain attributes [attribute name] If you want to select an element with a href attribute -> [href] {color: magenta;}
